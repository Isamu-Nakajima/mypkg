# ロボットシステム学課題２
##  概要
講義で作成したノードを改造し、ミニゲームを製作しました。
##  動作環境
- Ubuntu 20.04 LTS
- ROS version noetic
##  使用した道具
- Raspberry Pi 4
##  実行動画
<https://youtu.be/tOdbqqPCZ0U>
##  インストール方法
インストールは以下の手順で行ってください。
```
cd ~/catkin_ws/src
git clone https://github.com/Isamu-Nakajima/mypkg.git
cd ..
catkin_make
```
##  使用方法
```
roslaunch mypkg mypkg.launch
```
上記を実行後はプログラムに従ってください。
